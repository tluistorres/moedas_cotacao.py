## Cotação de Moedas.

Um aplicativo simples que exibe as cotações atuais de diferentes moedas em relação ao Real Brasileiro (BRL).

## Funcionalidades:

- Exibe as cotações atuais de moedas como USD, EUR, JPY, ARS, MXN, BTC, ETH e LTC
- Utiliza a API Awesome API para obter as cotações
- Exibe as cotações em tempo real

## Requisitos:

- Python 3.x
- Kivy
- requests

## Instalação:

1. Clone o repositório: git clone https://github.com/tluistorres/moedas_cotacao.py.git
2. Instale as dependências: pip install kivy requests
3. Execute o aplicativo: python app.py
